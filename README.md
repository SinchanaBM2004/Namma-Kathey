Namma-Kathey – Regional Hero Storybook App
Namma-Kathey (National Pride) is an interactive Android storybook app designed to introduce children to the local heroes, freedom fighters, poets, and social reformers of Karnataka.

The app builds local pride, patriotism, moral values, and linguistic heritage through engaging storytelling, quizzes, and interactive learning.

📌 Problem Statement
Children today know global superheroes but often lack awareness of their own district's historical heroes such as:

🛡️ Sangolli Rayanna
👑 Kittur Rani Chennamma
📜 Kuvempu
✊ Basavanna
There is a need for a digital, interactive, and child-friendly platform that narrates local bravery stories in simple language with visuals.

🌟 Vision
Namma-Kathey is a Regional Hero Storybook App that:

Promotes local pride
Encourages value-based education
Supports Kannada and English bilingual learning
Makes history interactive and fun
🚀 Features
🗺️ 1. Interactive District Map
Tap on any district of Karnataka
View the list of local heroes from that district
📖 2. Swipe-Based Illustrated Storybook
Built using ViewPager2
Smooth swipe navigation between story pages
Child-friendly illustrations
Simple language for better understanding
🔊 3. Read-Aloud Feature (Text-To-Speech)
Stories can be read aloud in:
✅ Kannada
✅ English
Helps:
Early readers
Visually impaired children
Pronunciation learning
🌐 4. Language Toggle (Mandatory Success Criteria ✅)
One-tap switch between:
Kannada
English
Dynamically updates:
Story text
Quiz questions
UI labels
🧠 5. Hero Quiz
3 simple multiple-choice questions per hero
Reinforces learning
Instant feedback
🏅 6. Heritage Badge System
Earn a Heritage Badge after completing quiz
Badge saved in:
✅ User Profile (Local storage / SharedPreferences / Room DB)
Encourages achievement and engagement
📍 7. Statue Finder
Shows:
Nearest statue/memorial of the hero
Uses:
Google Maps Intent / Location API
🎨 8. Child-Friendly UI
Vibrant colors
Large fonts
Cartoon-style illustrations
Simple navigation
🤖 AI Technologies Used
Namma-Kathey integrates Generative AI (GenAI) to enhance content creation and learning experience.
